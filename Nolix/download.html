How to import the Nolix library

Eclipse

  1. In the project explorer, right click on your project.
  2. Select 'Build Path'.
  3. Select 'Add External Archives'.

NetBeans

  1. In the project explorer, navigate to 'Libraries'.
  2. Right click on 'Libraries'.
  3. Select 'Add JAR/Folder'.
