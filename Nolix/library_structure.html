Main packages



ch.nolix.common Contains classes for basic purpose like containers or validators.
ch.nolix.element Contains Element classes. Elements are reproducable. That means that elements can be represented by Strings or specification files.
ch.nolix.techapi Contains APIs of technical features like complex numbers with dynamic size.
